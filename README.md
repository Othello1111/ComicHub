# comic-downloader

> Comic downloader (漫画下载器)

## 介绍

comic-downloader 是一款漫画下载器。爬取网站的图片，生成 PDF 文件。

Koa + MySQL + Umi + Electron + React + Typescript 打造 。

## 装包

前端装包

```bash
npm i
```

服务端装包

```bash
cd server
npm i
```

### 运行

启动前端页面

```bash
npm run start:react
```

启动服务端

```bash
npm run start:server
```

## 功能

1. [x] 下载一集漫画
2. [x] 下载整部漫画
3. [x] MySQL 存储爬取链接
4. [ ] 生成 PDF 文件
5. [ ] 前端交互页面
6. [ ] 打包成桌面应用
